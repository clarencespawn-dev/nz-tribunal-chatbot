# Screenshots

Drop your screenshots here, named to match what README.md references:

| Filename | What to capture |
|---|---|
| `chat-interface.png` | The main Streamlit chat screen — ideally with a question already asked and answered, so the page looks "alive" rather than empty |
| `answer-with-sources.png` | A question answered with the "Sources" expander opened, showing the cited tribunal page(s) and/or Act section(s) |
| `sidebar-settings.png` | The left sidebar showing the chunk count metric and the "Number of sources to retrieve" slider |
| `ingest-terminal.png` *(optional)* | Terminal output from running `python chatbot/ingest.py`, showing the chunk count being embedded |

## Tips for good screenshots

- Ask a real, specific question (e.g. *"What's the penalty for a landlord giving a retaliatory notice of termination?"*) rather than something generic — it shows the RAG pipeline actually retrieving relevant law
- Crop out your terminal prompt/username and any local file paths
- PNG format, reasonably sized (under ~500KB each keeps the repo light)
- 16:9 or similar landscape crop tends to render best in GitHub's README preview
